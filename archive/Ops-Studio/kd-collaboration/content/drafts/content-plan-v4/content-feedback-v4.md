## Design Enhancement Recommendations (For Discussion)

David mentioned wanting "80% more visual impact" ‚Äî here are concrete suggestions:

### High-Impact Changes

**1. Title Slide**
- Full-bleed image with dramatic overlay
- Deck title: 3‚Äì4x larger, bold serif or contrast font
- Quote: positioned artistically, not just centered
- Footer: subtle but present (bottom corner)

**2. Typography System**
- Heading font: bold serif (Playfair, Crimson, Lora) for gravitas
- Body font: clean sans (Inter, Epilogue, Satoshi) for readability
- Size scale: much more dramatic (h1 could be 48‚Äì72px)

**3. Color Enhancement**
- Current palette: cream, charcoal, gold, lavender (safe)
- Consider: richer jewel tones (deep plum, forest green, burnt amber) as accents
- Gold border treatment is working ‚Äî use more strategically
- Lavender CTA treatment is effective ‚Äî replicate elsewhere

**4. Slide-Specific Designs**
- **Alignment:** KD quote should be styled as pull quote (larger, indented, border)
- **Help:** Table should be cards or bordered sections, not plain rows
- **Regenerative:** Metrics should be styled as pills/badges, not plain list
- **Fit:** Three-column layout needs strong visual structure (cards? sections? borders?)
- **Engagement:** Cards need more depth (shadows, borders, maybe gradients)

**5. Spaciousness vs. Impact**
- Current design: spacious but safe
- Goal: maintain spaciousness but add bold moments
- Strategy: use contrast (one bold element per slide anchors the space)

### Design Prototyping Sequence

1. Start with Title slide (sets visual tone for entire deck)
2. Then Fit slide (most complex layout challenge)
3. Then Engagement cards (need more visual presence)
4. Then typography system across all slides
5. Finally, refinement pass on spacing/colors
